## Setup
This project use git and npm 

```git clone https://github.com/nguyenhoa2896/profile_project2.git```

```cd profile_project2```

```npm install```

```npm start```


To keep updated and get the latest commits from master

```git fetch origin```

```git rebase origin/master```
